# gardenlight

Einfach Gartenleute selber bauen mit dem Arduino Nano:

#### Funktionen

Button 1:
- white: block
- white: strip
- white: strip + block
- white: worm strip
- white: worm strip + block
- off (animation 1)

Button 2:
- red: red, orange
- orange: orange, yellow
- yellow: yellow, apple
- green: green, turquoise
- blue: blue, purple
- pink: pink, pinky
- off (animation 2)
